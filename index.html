0x0000 - Version

0x000C - Required shift in the X direction for orientation 1
0x000E - Required shift in the X direction for orientation 2
0x0010 - Required shift in the X direction for orientation 3
0x0012 - Required shift in the X direction for orientation 4
0x0014 - Required shift in the X direction for orientation 5
0x0016 - Required shift in the Y direction for orientation 0
0x0018 - Required shift in the Y direction for orientation 1
0x001A - Required shift in the Y direction for orientation 2
0x001C - Required shift in the Y direction for orientation 3
0x001E - Required shift in the Y direction for orientation 4
0x0020 - Required shift in the Y direction for orientation 5
0x0022 - Memory offset of first frame for direction 0 from beginning of frame area
0x0026 - Memory offset of first frame for direction 1 from beginning of frame area
0x002A - Memory offset of first frame for direction 2 from beginning of frame area
0x002E - Memory offset of first frame for direction 3 from beginning of frame area
0x0032 - Memory offset of first frame for direction 4 from beginning of frame area
0x0036 - Memory offset of first frame for direction 5 from beginning of frame area
0x003A - Size of frame area
0x003E - FRAME-WIDTH: Width of frame 0
0x0040 - FRAME-HEIGHT: Height of frame 0
0x0042 - FRAME-SIZE: Number of pixels for frame 0
0x0046 - Offset in X direction of frame 0
0x0048 - Offset in Y direction of frame 0
